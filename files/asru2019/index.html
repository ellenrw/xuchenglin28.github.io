Time-domain Speaker Extraction Network (TseNet)

ASRU2019: This webpage is to show some listening examples for the proposed TseNet method and other frequency-domain methods. The propose TseNet method can solve the inaccuracy phase estimation problem existed in frequency-domain methods. It also doesn't need the prior information of the number of speakers in the mixture, which is necessary in most source separation methods. As the number of speaker in the mixture is always unknown in real application. With the speaker characteristics, the proposed TseNet is practical to the real application where only the registered speaker is needed to be responded, such as, speaker verification where the target speaker is alway registered in advance, speech recognition for person devices.
